---
title: "生成图像"
description: "使用 POST /v1/images/generations 创建图像。"
keywords: ["图像生成", "Images API", "prompt"]
api: "POST /v1/images/generations"
---

根据文本提示生成图像。支持的尺寸、数量和返回格式由具体模型决定。

<ParamField body="model" type="string" required>
  支持图像生成的模型 ID。
</ParamField>

<ParamField body="prompt" type="string" required>
  描述目标图像的文本提示。
</ParamField>

<ParamField body="n" type="integer" default="1">
  要生成的图像数量。部分模型只支持 `1`。
</ParamField>

<ParamField body="size" type="string">
  图像尺寸。可用值由模型决定。
</ParamField>

<RequestExample>
```json
{
  "model": "IMAGE_MODEL_ID",
  "prompt": "蓝色数据流穿过云端 API 网关，简洁的等距插画",
  "n": 1,
  "size": "1024x1024"
}
```
</RequestExample>

<ResponseExample>
```json
{
  "created": 1782972000,
  "data": [
    {
      "url": "https://example-cdn.invalid/generated-image.png"
    }
  ]
}
```
</ResponseExample>

<Note>
  返回值可能是临时 URL 或 Base64 数据。请根据实际响应及时下载并保存生成结果。
</Note>
