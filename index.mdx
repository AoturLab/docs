## 最小调用示例

<CodeGroup>
```python Python
from openai import OpenAI

client = OpenAI(
    api_key="YOUR_HILINKUP_API_KEY",
    base_url="https://hilinkup.com/v1",
)

response = client.chat.completions.create(
    model="qwen3.7-plus",
    messages=[
        {"role": "user", "content": "用一句话介绍 HiLinkup"}
    ],
)

print(response.choices[0].message.content)
```

```javascript Node.js
import OpenAI from "openai";

const client = new OpenAI({
  apiKey: process.env.HILINKUP_API_KEY,
  baseURL: "https://hilinkup.com/v1",
});

const response = await client.chat.completions.create({
  model: "qwen3.7-plus",
  messages: [
    { role: "user", content: "用一句话介绍 HiLinkup" },
  ],
});

console.log(response.choices[0].message.content);
```

```bash cURL
curl https://hilinkup.com/v1/chat/completions \
  -H "Authorization: Bearer $HILINKUP_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "qwen3.7-plus",
    "messages": [
      {"role": "user", "content": "用一句话介绍 HiLinkup"}
    ]
  }'
```
</CodeGroup>

<Tip>
  模型可用性可能随账户和渠道变化。发送请求前，通过
  [`GET /v1/models`](/api-reference/models) 获取当前 API Key 可用的模型 ID。
</Tip>

## 推荐接入顺序

<Steps>
  <Step title="创建 API Key">
    登录 [HiLinkup 控制台](https://hilinkup.com/console/token)，为每个应用创建独立密钥。
  </Step>
  <Step title="获取模型">
    调用 `/v1/models`，不要长期硬编码未经验证的模型列表。
  </Step>
  <Step title="完成基础调用">
    先测试普通对话，再验证流式响应、工具调用和多模态输入。
  </Step>
  <Step title="接入生产环境">
    配置超时、有限重试、日志脱敏、额度和异常告警。
  </Step>
</Steps>
