---
title: "Chatbox"
description: "在 Chatbox 中添加 HiLinkup 自定义 OpenAI 服务。"
keywords: ["Chatbox", "聊天客户端", "自定义模型"]
icon: "message-circle"
---

<Steps>
  <Step title="添加模型提供方">
    打开 Chatbox 设置，选择自定义 OpenAI API 或 OpenAI-compatible。
  </Step>
  <Step title="填写连接信息">
    - API Host：`https://hilinkup.com/v1`
    - API Key：HiLinkup API Key
    - Model：当前可用的模型 ID
  </Step>
  <Step title="测试对话">
    保存后新建会话，确认普通响应和流式响应均正常。
  </Step>
</Steps>

<Tip>
  为 Chatbox 创建独立密钥。设备丢失或不再使用该客户端时，可以单独撤销，不影响服务器应用。
</Tip>
