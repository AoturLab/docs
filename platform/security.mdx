## API Key

- 仅保存在服务端环境变量或密钥管理服务。
- 不嵌入网页、移动端安装包或桌面客户端。
- 按应用和环境隔离密钥。
- 定期轮换，泄露后立即撤销。

## 数据和日志

- 只发送完成任务所需的数据。
- 在日志中脱敏 API Key、个人信息和业务机密。
- 为提示词和模型输出设置保留周期。
- 联系支持时仅提供请求 ID，不提供完整密钥。

## 工具调用

- 将模型输出视为不可信输入。
- 在服务端重新检查用户身份和权限。
- 对付款、删除和外发操作增加确认。
- 限制工具可访问的网络、文件和账户范围。

<Warning>
  模型生成了合法 JSON，不代表该操作已经获得授权。参数校验和业务授权必须由你的服务端完成。
</Warning>
