Claude Code uses Anthropic-style requests. Use this page only when your HiLinkup account and target model support [Claude Messages](/en/api-reference/claude-messages)-compatible calls.

<Note>
  Examples use the HiLinkup platform model `deepseek-v3`. This is a Chinese model example, not a requirement to use an overseas Claude model.
</Note>

## Recommended: user settings

Claude Code official settings use `~/.claude/settings.json` for user settings. On Windows, `~/.claude` resolves to `%USERPROFILE%\.claude`.

<Tabs>
  <Tab title="macOS">
    ```bash
    mkdir -p ~/.claude
    nano ~/.claude/settings.json
    ```

    Paste:

    ```json
    {
      "$schema": "https://json.schemastore.org/claude-code-settings.json",
      "env": {
        "ANTHROPIC_AUTH_TOKEN": "YOUR_HILINKUP_API_KEY",
        "ANTHROPIC_BASE_URL": "https://hilinkup.com",
        "ANTHROPIC_MODEL": "deepseek-v3"
      }
    }
    ```

    Save with `Ctrl + O`, press Enter, then exit with `Ctrl + X`.
  </Tab>
  <Tab title="Linux">
    ```bash
    mkdir -p ~/.claude
    nano ~/.claude/settings.json
    ```

    Paste:

    ```json
    {
      "$schema": "https://json.schemastore.org/claude-code-settings.json",
      "env": {
        "ANTHROPIC_AUTH_TOKEN": "YOUR_HILINKUP_API_KEY",
        "ANTHROPIC_BASE_URL": "https://hilinkup.com",
        "ANTHROPIC_MODEL": "deepseek-v3"
      }
    }
    ```

    Save with `Ctrl + O`, press Enter, then exit with `Ctrl + X`.
  </Tab>
  <Tab title="Windows PowerShell">
    ```powershell
    New-Item -ItemType Directory -Force "$env:USERPROFILE\.claude"
    notepad "$env:USERPROFILE\.claude\settings.json"
    ```

    Paste and save:

    ```json
    {
      "$schema": "https://json.schemastore.org/claude-code-settings.json",
      "env": {
        "ANTHROPIC_AUTH_TOKEN": "YOUR_HILINKUP_API_KEY",
        "ANTHROPIC_BASE_URL": "https://hilinkup.com",
        "ANTHROPIC_MODEL": "deepseek-v3"
      }
    }
    ```
  </Tab>
</Tabs>

## Project-local settings

Use `.claude/settings.local.json` when only the current project should use HiLinkup.

<Tabs>
  <Tab title="macOS / Linux">
    ```bash
    mkdir -p .claude
    nano .claude/settings.local.json
    ```
  </Tab>
  <Tab title="Windows PowerShell">
    ```powershell
    New-Item -ItemType Directory -Force ".claude"
    notepad ".claude\settings.local.json"
    ```
  </Tab>
</Tabs>

Paste:

```json
{
  "env": {
    "ANTHROPIC_AUTH_TOKEN": "YOUR_HILINKUP_API_KEY",
    "ANTHROPIC_BASE_URL": "https://hilinkup.com",
    "ANTHROPIC_MODEL": "deepseek-v3"
  },
  "permissions": {
    "deny": [
      "Read(./.env)",
      "Read(./.env.*)",
      "Read(./secrets/**)"
    ]
  }
}
```

Do not commit `.claude/settings.local.json`.

## Temporary environment variables

<Tabs>
  <Tab title="macOS / Linux">
    ```bash
    export ANTHROPIC_AUTH_TOKEN="YOUR_HILINKUP_API_KEY"
    export ANTHROPIC_BASE_URL="https://hilinkup.com"
    export ANTHROPIC_MODEL="deepseek-v3"
    claude
    ```
  </Tab>
  <Tab title="Windows PowerShell">
    ```powershell
    $env:ANTHROPIC_AUTH_TOKEN = "YOUR_HILINKUP_API_KEY"
    $env:ANTHROPIC_BASE_URL = "https://hilinkup.com"
    $env:ANTHROPIC_MODEL = "deepseek-v3"
    claude
    ```
  </Tab>
</Tabs>

## Verify

First test the protocol directly:

<Tabs>
  <Tab title="macOS / Linux">
    ```bash
    curl https://hilinkup.com/v1/messages \
      -H "content-type: application/json" \
      -H "x-api-key: $HILINKUP_API_KEY" \
      -H "anthropic-version: 2023-06-01" \
      -d '{
        "model": "deepseek-v3",
        "max_tokens": 256,
        "messages": [
          {
            "role": "user",
            "content": "Reply with: HiLinkup Claude route is working."
          }
        ]
      }'
    ```
  </Tab>
  <Tab title="Windows PowerShell">
    ```powershell
    curl.exe https://hilinkup.com/v1/messages `
      -H "content-type: application/json" `
      -H "x-api-key: $env:HILINKUP_API_KEY" `
      -H "anthropic-version: 2023-06-01" `
      -d '{
        "model": "deepseek-v3",
        "max_tokens": 256,
        "messages": [
          {
            "role": "user",
            "content": "Reply with: HiLinkup Claude route is working."
          }
        ]
      }'
    ```
  </Tab>
</Tabs>

Then start Claude Code and send a plain prompt before testing file reads, shell commands, or long agent tasks.

<Warning>A working OpenAI Chat Completions call does not prove the Anthropic Messages route is available. Confirm the model and protocol in the HiLinkup console.</Warning>
