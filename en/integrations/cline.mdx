---

Select an OpenAI-compatible provider in Cline and enter:

- Base URL: `https://hilinkup.com/v1`
- API key: your HiLinkup API key
- Model ID: a value returned by `/v1/models`

Start with a read-only task, then validate tool calling before allowing file or command actions.
