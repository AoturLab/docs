Add a chat model using an OpenAI-compatible provider:

```text
API key:  YOUR_HILINKUP_API_KEY
Base URL: https://hilinkup.com/v1
Model:    an exact ID from /v1/models
```

Save the configuration and ask Continue to explain a short selected code block. Configure chat, edit, autocomplete, and agent models separately; one model is not necessarily suitable for every protocol.

Store credentials in environment variables or private user settings, not in a shared repository configuration.
