In Dify model provider settings, add an OpenAI-compatible model:

- API endpoint: `https://hilinkup.com/v1`
- API key: your HiLinkup API key
- Model name: an ID returned by `/v1/models`

Test chat, streaming, and tool behavior from the workflow that will use the model.
