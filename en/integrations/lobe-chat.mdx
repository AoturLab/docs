---

Configure the OpenAI-compatible provider in LobeChat with:

```text
API key:  YOUR_HILINKUP_API_KEY
Base URL: https://hilinkup.com/v1
```

Add the required model IDs to the model list. For server deployments, store the API key in the deployment secret manager.
