```python
from langchain_openai import ChatOpenAI

model = ChatOpenAI(
    api_key="YOUR_HILINKUP_API_KEY",
    base_url="https://hilinkup.com/v1",
    model="qwen3.7-plus",
)

print(model.invoke("Hello").content)
```

Use environment variables for credentials and add timeouts and bounded retries for production.
