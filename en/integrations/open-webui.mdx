---
title: "Open WebUI"
description: "Connect Open WebUI to HiLinkup."
keywords: ["Open WebUI", "connection", "models"]
icon: "panel-top"
---

Add an OpenAI API connection in the administrator settings:

- URL: `https://hilinkup.com/v1`
- Key: your HiLinkup API key

Refresh the model list and restrict shared deployments to approved models and users.

<Warning>Do not expose the API key through browser-side configuration on a public deployment.</Warning>
