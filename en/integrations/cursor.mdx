In Cursor model settings, add an OpenAI-compatible provider:

- API key: your HiLinkup API key
- Base URL: `https://hilinkup.com/v1`
- Model: an ID from `/v1/models`

Test with a short chat request. If the model is rejected, verify the exact ID and account access.
