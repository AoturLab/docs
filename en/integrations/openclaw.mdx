Create an OpenAI-compatible provider using:

```text
Base URL: https://hilinkup.com/v1
API key:  YOUR_HILINKUP_API_KEY
Model:    a model ID from /v1/models
```

After saving, test the full agent path with the same tools and prompt size used in production. A basic API request alone does not validate the complete agent workflow.
