---
title: "Authentication"
description: "Authenticate HiLinkup API requests securely."
keywords: ["authentication", "Bearer token", "API key"]
icon: "key-round"
---

Send your API key as a Bearer token on every request.

```http
Authorization: Bearer YOUR_HILINKUP_API_KEY
```

Create separate keys for different applications and environments. Store keys in environment variables or a secret manager, never in source code.

<Warning>If a key is exposed, revoke it in the console and create a replacement immediately.</Warning>

See [error troubleshooting](/en/troubleshooting/errors) for `401` and `403` responses.
