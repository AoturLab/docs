---
title: "Security"
description: "Protect API keys and sensitive model data."
keywords: ["security", "API keys", "privacy"]
icon: "shield-check"
---

- Store API keys in a secret manager
- Use separate keys for development and production
- Rotate exposed or unused keys
- Redact credentials and sensitive prompts from logs
- Apply least privilege to tools called by models

<Warning>Never send an API key to browser code or commit it to a repository.</Warning>
