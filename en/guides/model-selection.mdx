Start with the models returned by [`GET /v1/models`](/en/api-reference/models). Evaluate candidates with representative prompts.

Compare:

- Output quality and instruction following
- Latency and streaming behavior
- Context-window and tool-call support
- Cost and availability

Do not assume that a model visible to one API key is available to every account.
