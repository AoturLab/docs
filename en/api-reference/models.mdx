## `GET /v1/models`

```bash
curl https://hilinkup.com/v1/models \
  -H "Authorization: Bearer $HILINKUP_API_KEY"
```

Use the returned `data[].id` value as the `model` field in subsequent requests.

<Note>The response is the source of truth for the current API key. Availability can change over time.</Note>
