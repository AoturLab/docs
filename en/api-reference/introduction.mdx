HiLinkup exposes OpenAI-compatible endpoints and selected provider-style compatible endpoints. Most OpenAI-compatible clients only need a new API key, base URL, and model ID. Claude-style and Gemini-style clients must use their own endpoint paths and auth headers.

## Base URLs

OpenAI-compatible endpoints:

```text
https://hilinkup.com/v1
```

Gemini-compatible endpoints use the `v1beta` path:

```text
https://hilinkup.com/v1beta/models/{model}:generateContent
```

## Authentication

OpenAI-compatible endpoints:

```http
Authorization: Bearer YOUR_HILINKUP_API_KEY
Content-Type: application/json
```

Claude Messages-compatible endpoints:

```http
x-api-key: YOUR_HILINKUP_API_KEY
anthropic-version: 2023-06-01
Content-Type: application/json
```

Gemini-compatible endpoints:

```http
x-goog-api-key: YOUR_HILINKUP_API_KEY
Content-Type: application/json
```
