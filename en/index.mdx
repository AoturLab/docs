HiLinkup provides a unified, OpenAI-compatible model API. In most applications, you only need to replace the API key, base URL, and model ID.

<Info>API base URL: `https://hilinkup.com/v1`</Info>

## Minimal example

```python
from openai import OpenAI

client = OpenAI(
    api_key="YOUR_HILINKUP_API_KEY",
    base_url="https://hilinkup.com/v1",
)

response = client.chat.completions.create(
    model="qwen3.7-plus",
    messages=[{"role": "user", "content": "Hello"}],
)
print(response.choices[0].message.content)
```

<Tip>Call [`GET /v1/models`](/en/api-reference/models) before choosing a model. Availability can vary by account and provider channel.</Tip>
